# Data Structures and Algorithms Projects

This repository contains implementations of fundamental data structures and numerical libraries in C++.

Projects included:
- Big Integer Library
- Deque Implementation

---

# 1. Big Integer Library

A custom implementation of arbitrary precision arithmetic for handling very large integers beyond standard data type limits.

## Features

- Supports very large integers
- Addition
- Subtraction
- Multiplication
- Division
- Modulus operations
- Operator overloading
- Negative number handling

## Concepts Used

- Dynamic memory management
- String/array based arithmetic
- Operator overloading
- OOP principles

---

# 2. Deque Implementation

A custom implementation of a Double Ended Queue (Deque) in C++.

## Features

- Insert from front and rear
- Delete from front and rear
- Dynamic resizing
- Efficient push/pop operations
- Random access support

## Concepts Used

- Circular array / dynamic array
- Memory management
- Class design
- Time complexity optimization
