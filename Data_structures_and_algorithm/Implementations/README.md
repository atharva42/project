# Data Structures and Algorithms Projects

This repository contains custom implementations of fundamental data structures and arbitrary-precision arithmetic libraries in C++.

## Projects

* Big Integer Library
* Deque Implementation

---

# 1. Big Integer Library

A custom arbitrary-precision arithmetic library that performs mathematical operations on integers of virtually unlimited size using string-based computation.

## Features

### Core Arithmetic

* Addition
* Subtraction
* Multiplication
* Division
* Modulus

### Mathematical Operations

* Greatest Common Divisor (GCD)
* Exponentiation (Exponentiation by Squaring)
* Factorial Computation

### Expression Processing

* Infix to Postfix Conversion
* Postfix Expression Evaluation
* Arithmetic Expression Calculator
* Multi-digit Operand Support
* Operator Precedence Handling

### Supported Operators

* Addition (`+`)
* Subtraction (`-`)
* Multiplication (`x`)
* Division (`/`)

## Algorithms Used

* Long Addition
* Long Subtraction
* Long Multiplication
* Long Division
* Euclidean Algorithm
* Exponentiation by Squaring
* Stack-Based Expression Evaluation

# 2. Deque Implementation

A custom implementation of a Double Ended Queue (Deque) using a dynamically resizable circular array.

## Features

### Constructors

* Default Constructor
* Size Constructor
* Fill Constructor

### Insertion & Deletion

* Push Front
* Push Back
* Pop Front
* Pop Back

### Element Access

* Front Element Access
* Back Element Access
* Index-Based Access (`operator[]`)
* Negative Indexing Support

### Capacity Management

* Dynamic Resizing
* Reserve Capacity
* Shrink to Fit
* Clear Deque
* Capacity Query

### Utility Operations

* Empty Check
* Size Query

## Data Structure Concepts

* Circular Array (Ring Buffer)
* Dynamic Memory Allocation
* Template Programming
* Amortized Capacity Expansion
